# Express + MySQL

try to use MySQL with Nodejs 

## Get started

- Clone this repository
- Create `.env` like `.env.example` enter your MySQL -- $!important$ --
- `npm install`
- `npm run dev`

## Inside it

- express
- dotenv
- mysql
- nodemon
